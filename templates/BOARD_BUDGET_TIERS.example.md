# Board Budget Tiers

Use this as a discussion scaffold, not as a final quote.

## Minimum durable improvement

- replace the weakest storage or backup path
- stabilize the docs mirror and known-good archive
- keep the assistant node and edge nodes intact

## Recommended central improvement

- a modern central server or workstation with reliable storage
- enough RAM for the spine services and archive
- enough headroom to keep the room boring

## Long-term ideal

- central server with comfortable capacity
- room for future growth without changing the architecture
- enough spare headroom that maintenance does not depend on luck

- write the final numbers in a separate budget sheet
- keep this file as a structure for the conversation
