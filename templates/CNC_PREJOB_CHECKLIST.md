# CNC Pre-Job Checklist

Use before opening/sending a job through CNCjs.

- [ ] Adult/operator present
- [ ] Correct bit installed
- [ ] Workpiece clamped
- [ ] Toolpath reviewed
- [ ] Material and bit match
- [ ] Zero confirmed
- [ ] Dust/debris path clear
- [ ] Emergency stop / stop procedure known
- [ ] No loose items near machine
- [ ] CNCjs connected to the expected controller
