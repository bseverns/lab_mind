# Day 0 Prep

## Before the rebuild day
Prepare these before anyone inserts a blank card:

- flashed microSD card(s)
- printed or tablet copy of this docs set
- office Wi-Fi or Ethernet details
- one confirmed password vault or sealed note for local credentials
- Headscale details if available
- Digital Factory login bookmark
- list of expected printers
- one known-good power setup

## Decide now
Fill or confirm these before rebuild day if possible:

- hostname for Jetson-A
- SSD mount path
- Docker data-root path
- whether Cubiko will be attached locally
- whether Headscale enrollment will use preauth key or interactive flow
- who is allowed remote access
