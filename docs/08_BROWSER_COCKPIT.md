# Browser Cockpit

## Summary
The browser should be the main operator surface.

## Pinned tabs
1. Digital Factory
2. Open WebUI
3. Portainer
4. Node-RED
5. Syncthing
6. CNCjs if enabled

## Why
A browser-first cockpit means:
- easier handoff to interns or educators
- less need for full desktop remote control
- fewer accidental CLI-only recovery bottlenecks
