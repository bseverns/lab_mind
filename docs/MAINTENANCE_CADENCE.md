# Maintenance Cadence

This page says how often to look at the important stuff.
Keep the cadence boring.

## Cadence

| Interval | Task |
|---|---|
| Daily or class-day | cockpit check, assistant check, printer visibility |
| Weekly | backup check, incident review, inventory check |
| Monthly | restore drill, known-good-state capture, model sanity check |
| Quarterly | policy review, optional surfaces review, naming review |

## What to keep current

- tested values
- image versions
- current state
- known-good state
- ownership map

## Rule

If a doc has not been touched in months, treat it as suspect until it is re-verified.
