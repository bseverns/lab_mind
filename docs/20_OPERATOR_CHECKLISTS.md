# Operator Checklists

## Daily quick check
- machine is on
- network is up
- browser cockpit opens
- Digital Factory opens
- expected printer fleet appears
- no surprise alerts

## Weekly check
- reboot if appropriate
- Docker status check
- storage usage check
- backup presence check
- update log note
- print or CNC status notes if relevant

## After an incident
- write incident note
- write what actually fixed it
- update docs if the fix revealed a missing step
