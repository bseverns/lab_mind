# Open WebUI and Ollama

## Role
This is the assistant surface.

## Minimum success signal
- Ollama responds locally
- Open WebUI loads in the browser
- a small model is available
- the box does not thrash or freeze under ordinary use

## Default posture
Prefer smaller models as defaults.
Use larger experiments only when they clearly justify the load.

## Storage note
Model blobs belong on NVMe-backed storage, not on the microSD card.
