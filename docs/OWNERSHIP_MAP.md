# Ownership Map

This page answers who is responsible for what.
It is intentionally small and practical.

## Ownership

| Area | Owner | Notes |
|---|---|---|
| R900 spine | `EDIT_ME` | backups, storage, docs mirror, Portainer |
| Jetson-A assistant | `EDIT_ME` | local model and browser assistant |
| Edge nodes | `EDIT_ME` | room status and support roles |
| Headscale values | `EDIT_ME` | private access control |
| Printer inventory | `EDIT_ME` | Digital Factory alignment |
| CNC path | `EDIT_ME` | only when physically adjacent |

## What this means

- The spine should have a clear steward.
- Model changes should not be made casually.
- Private access values should have a single accountable owner.
- Edge devices may be replaceable, but their role still needs ownership.

## Rule

If nobody owns it, it will drift.
