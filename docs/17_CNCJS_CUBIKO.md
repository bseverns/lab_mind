# CNCjs and Cubiko

## Summary
Use this only if the Cubiko is physically near Jetson-A.

## Preconditions
- stable USB connection
- local shop-safe placement
- browser access to CNCjs
- clear operator understanding that the dashboard is not a substitute for being present and attentive

## Success signal
- CNCjs opens
- machine status is visible
- no dangerous improvisation is happening because the dashboard exists
