# Hardware Layout

## Primary machine
- Jetson Orin Nano dev kit
- microSD boot medium
- NVMe SSD for durable data
- network connection to office LAN
- monitor / keyboard / mouse available for first-day setup

## Optional peripherals
- UPS if available
- label printer or tape labels for hostname / IP notes
- spare microSD cards
- spare USB keyboard
- spare Ethernet cable
- notebook or printed checklist

## Physical layout rule
Do not place the Jetson where it must be moved every day in order to access power, Ethernet, or the Cubiko USB line.

## Suggested labels
Physically label:
- device name: `JETSON-A`
- power brick
- microSD card version/date
- SSD mount note
- browser cockpit QR or short URL if you create one
