# System Purpose

## What this machine is
Jetson-A is the studio / office helper box.

It should do a few jobs well:

- serve a local assistant surface
- host browser-first tools for operations
- provide private remote access for educators
- act as the local companion to Digital Factory
- optionally host `cncjs` if the Cubiko is physically nearby

## What it is not
It is not:

- a public internet application
- a replacement for Digital Factory
- a general office NAS
- a full remote-office VPN for everyone
- a magic AI brain that should run every possible service at once

## Success signal
The box is doing its job when:

- it boots reliably
- it survives a blank-card rebuild without drama
- educators can reach it privately from outside the room
- its key tools open in a browser
- it does not become a second full-time job
