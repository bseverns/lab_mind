# Security Posture

## Summary
The lab-brain stack should be private, boring, and replaceable.

## Principles
- private operator access over Headscale-managed path
- no public exposure of internal browser tools by default
- smallest practical service set
- document what changes
- backups off the box
- no guessing of credentials or enrollment methods

## This box is not
- the public face of the organization
- a general office VPN concentrator
- the place to test every cool service you find online
