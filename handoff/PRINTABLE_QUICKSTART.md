# Printable Quickstart

1. Open `http://r900.local`.
2. If you need the assistant, open `http://jetson-a.local:3000`.
3. Confirm Digital Factory matches the inventory template.
4. Use Open WebUI for assistant help.
5. Use Portainer only if you have operator permission.
6. Use CNCjs only after the physical checklist.
7. Use the operator toggle checklist if code mode needs to change.
8. Use the model switch card if the Jetson-A profile needs to change.
9. Use the model choice ladder if you need the shortest version.
10. Record incidents. Do not guess secrets or network values.
